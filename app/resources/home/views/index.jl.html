Productivity of Rails. At the speed of C.

Genie is a full stack web framework that empowers developers to build fast, scalable, feature rich web applications in a highy productive way.

Building on top of Julia -- a new easy to learn, blazing fast programming language developed at MIT -- Genie provides a unique combination of features and a complete toolset that can be used throughout the full software stack.

Genie enables developers to use a single programming language to write DRY reusable code, from web apps, to REST APIs, to Data Science and AI powered real time analytics, to command line scripts. From MVP to millions of users.
